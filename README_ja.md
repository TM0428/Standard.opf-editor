# epub metadata editor

standard.opf の編集を楽にするツール

![](image/2023-12-08_20h05_41.png)

## 変更可能なメタデータ

-   Title
-   Author
-   Publisher
-   Synopsis
-   Identifier
-   Language
-   Series

## Install library

```command
pip install -r requirements.txt
```

## Excute

```command
python3 src/main.py
```
